HTML Attributes

All HTML elements can have Attributes.
Attributes provides additional information about elements.
Attributes are always specified in the start tag.
Attributes usually come in name to value pairs like: name="value".


href Attributes

The a tag defines a hyperlink while
the href attribute specifies the url of the page goes to.

Visit W3schools!

src Attributes

the img tag is used to embed image while
the src attribute specifies the path of the image embedded.

there are 2 ways to specify the URL in src attributes.
1. Absolute URL - Links to an external image that is hosted on another website.
2. Relative URL - Links to an image that is hosted within the website.

Tips!
if the URL begins with a slash, /, it will be an Absolute URL and vice versa.

width and height attributes

the img tag SHOULD also contain width and height attributes.

alt Attributes

The alt attributes specifies an alternate text for an image,
in the times that the image failed to be displayed.

style Attributes

The style attributes is used to add styles to an elements, such
as color, font, size and more.
